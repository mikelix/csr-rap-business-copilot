# Contributing

感谢您关注 CSR-RAP Business Copilot。

目前项目处于第一代 MVP 验证阶段。建议先通过 GitHub Issue 提交：

- 产品问题或界面缺陷
- 评分逻辑问题
- 简体中文表达改进
- 手机端兼容性问题
- 数据隐私或安全问题

在获得项目维护者确认前，请不要提交涉及 CSR-RAP 理论核心、商业三性知识产权或大规模架构重写的 Pull Request。

提交代码前请运行：

```bash
npm run lint
npm run build
```

请勿在 Issue、Pull Request 或代码中提交企业秘密、客户资料、API 密钥、账号密码或其他敏感信息。
